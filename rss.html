RSS (Really Simple Syndication) parsing library for erlang.

Authors: "Jeremy (Zaphar) Wall (jeremy@marzhillstudios.com).

Description

RSS (Really Simple Syndication) parsing library for erlang. This library does not parse RDF which is sometimes mistaken for RSS.

To get the record specifications include the include/rss.hrl header file in your code.
